# Registros de Decisión de Arquitectura (ADR)

Los ADR se publican automáticamente en nuestra base de conocimientos de arquitectura Log4brains:

🔗 **https://tsorren.github.io/DonaTrack-TP-DDS/adr-preview**

Por favor, utiliza este enlace para navegar por ellos.

## Desarrollo

Si aún no lo has hecho, instala Log4brains:

```bash
npm install -g log4brains
```

Para previsualizar la base de conocimientos localmente, ejecuta:

```bash
log4brains preview
```

En el modo de previsualización, la función de Hot Reload está activada: cualquier cambio que realices en un archivo
markdown se aplicará en vivo en la interfaz.

Para crear un nuevo ADR de forma interactiva, ejecuta:

```bash
log4brains adr new
```
